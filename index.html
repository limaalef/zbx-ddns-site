Zabbix Proxy DDNS

Registre um subdomínio gratuito e configure em seu proxy Zabbix para que ele continue se comunicando com sua infraestrutura mesmo em ambientes com IP dinâmico.

1
Registre seu subdomínio
.seudominio.com

Use apenas letras minúsculas, números e hífen.

  • Faça login para ver seus subdomínios registrados.
2
Script de atualização de IP
Esse script roda no seu servidor e mantém o registro DNS atualizado.

Selecione um subdomínio na lista acima para gerar o script com seus dados.

ddns_update.sh
3
Configurando no Linux
Passo a passo para instalar, agendar e manter o script funcionando.

3.1 Criar a pasta e salvar o script

Crie um diretório dedicado e salve o script do passo anterior nele:

bash

3.2 Ajustar permissões

Garanta que apenas o seu usuário possa ler e executar o script (ele contém sua chave de acesso):

bash

3.3 Testar manualmente

Execute uma vez para confirmar que está tudo certo antes de agendar:

bash

3.4 Configurar o cron

Agende a execução periódica (a cada 5 minutos). O rate limit do servidor permite até 5 execuções na primeira hora e depois 1 a cada 15 minutos, então um intervalo de 5 em 5 minutos é seguro mesmo que o IP não mude:

crontab -e

3.5 Verificar o log

O script registra apenas erros e mudanças de IP (log enxuto), para acompanhar:

bash

3.6 Remover (desinstalar)

Para parar de atualizar e remover os arquivos locais (isso não exclui o subdomínio nem o registro DNS — para isso, use o botão de exclusão na lista acima):

bash
